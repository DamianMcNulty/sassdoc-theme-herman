colors

brand-colors

We use bright primary colors for the main brand, everything else is based on these colors.

brand-orange #ff8b3d
brand-blue #4cc7f0
brand-pink #f04c9e
$brand-colors: (
  'brand-orange': hsl(24, 100%, 62%),
  'brand-blue': hsl(195, 85%, 62%),
  'brand-pink': hsl(330, 85%, 62%),
);

accent-colors

We have a set of secondary accent colors to help flesh out the palette.

lightest #fff
lighter #ebecec
light #b8bdbf
dark #1b4654
go #bedd5f
yield #f0c74c
stop #f04c4c
$accent-colors: (
  'lightest': #fff,
  'lighter': 'brand-blue' ('tint': 80%, 'desaturate': 80%),
  'light': 'brand-blue' ('tint': 30%, 'desaturate': 80%),
  'dark': 'brand-blue' ('shade': 65%),
  'go': hsl(75, 65%, 62%),
  'yield': hsl(45, 85%, 62%),
  'stop': hsl(0, 85%, 62%),
);

theme-colors

Applying the brand and accent colors to a sitewide theme.

contrast-light #fff
contrast-dark #1b4654
background #fff
text #1b4654
border #b8bdbf
callout #ebecec
overlay rgba(235,236,236,0.9)
shadow #b8bdbf
action #c75000
focus #b34800
active #c75000
accent #e2127a
$theme-colors: (
  'contrast-light': 'lightest',
  'contrast-dark': 'dark',
  'background': 'lightest',
  'text': 'dark',
  'border': 'light',
  'callout': 'lighter',
  'overlay': 'callout' ('rgba': 0.9),
  'shadow': 'light',
  'action': 'brand-orange' ('darken': 23%),
  'focus': 'action' ('shade': 10%),
  'active': 'action',
  'accent': 'brand-pink' ('darken': 14%),
);
